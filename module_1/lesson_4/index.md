# Числа
В JS можно использовать ***е***, которое заменяет кол-во нулей, т.е.  
1e9 = 1 000 000 000  
1e-6 = 0.000001  
Метод ***num.toString(base)*** возвращает строковое представление числа ***num*** в системе счисления ***base***.  
### Округление
* Math.floor - округление в меньшую сторону
* Math.ceil - округление в большую сторону
* Math. round - округление до ближайшего целого
* Number.toFixed(n) - округляет число до n знаков после запятой и возвращает строку
### Функции для работы с числами
* parseInt('123') - преобразует в число
* parseFloat('123.1') - преобразует в число с плав.точкой
* Math.random() -  возвращает псевдорандомное число в диапазоне от 0 до 1
* Math.max(1,2,3) Math.min(1,2,3) - возвращает максимальное/минимальное число
* Math.pow(n, power) - возвращает число, возведённое в ступень power
